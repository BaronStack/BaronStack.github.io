今天开始为深度学习的基础做准备，遇到python的一个编码问题：

python解释器脚本默认编码是ASCII码的形式，而你却输入了汉字，即将文件格式转换为utf-8的形式了，当下的解释器无法正确翻译这样的格式

解决办法：(在程序源文件开头 加上以下任意一个转码方式)

  • #coding=uft-8

  • #_*coding:UTF-8\*_

这样就可以愉快得输出想要的结果了
